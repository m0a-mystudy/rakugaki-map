# 落書きマップ (Rakugaki Map)

地図上に描画して友人と共有できるWebアプリケーションです。

## 機能

- 🗺️ Google Maps上での描画
- ✏️ 複数の描画ツール（ペン、直線、四角形、円）
- 🎨 カラーパレット（8色）
- 📏 線の太さ調整
- 💾 描画データの保存（Firebase Firestore）
- 🔗 共有リンク生成
- 📱 レスポンシブデザイン
- 🔒 セキュアなアクセス制御（読み取り全員、編集は認証済みユーザーのみ）

## 使い方

1. アプリケーションにアクセス
2. 地図上の任意の場所で描画ツールを使って絵を描く
3. 「保存」ボタンで描画を保存
4. URLを共有して他の人に公開
